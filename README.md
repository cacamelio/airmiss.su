# <h3>Prerequisites</h3>

# <ul>

# &nbsp; <li><b>IDE:</b> Visual Studio 2019 or 2022</li>

# &nbsp; <li><b>Workloads:</b> Desktop development with C++</li>

# &nbsp; <li><b>SDKs:</b> DirectX SDK (June 2010)</li>

# </ul>

# 

# <h3>Installation</h3>

# <ol>

# &nbsp; <li>Clone the repository:

# &nbsp;   <pre><code>git clone https://github.com/cacamelio/airmiss.us.git</code></pre>

# &nbsp; </li>

# &nbsp; <li>Open the <code>.sln</code> file in Visual Studio.</li>

# &nbsp; <li>Set the build configuration to <b>Release | x86</b>.</li>

# &nbsp; <li>Build the solution (<code>Ctrl + Shift + B</code>).</li>

# &nbsp; <li>Inject the resulting <code>.dll</code> into <code>csgo.exe</code> using your preferred injector (e.g., Process Hacker, CSGhost).</li>

# </ol>
